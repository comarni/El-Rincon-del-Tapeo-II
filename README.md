# El Rincón del Tapeo II - Sitio Web

Sitio web oficial del bar "El Rincón del Tapeo II" ubicado en Alcobendas, Madrid.

## 🏢 Sobre el Proyecto

Página web moderna y responsive para un bar de tapas tradicional español. Diseñada para mostrar la carta, información de contacto, horarios y permitir reservas online.

## 🚀 Características

- ✅ **Diseño completamente responsive** (mobile, tablet, desktop)
- ✅ **Navegación suave** entre secciones
- ✅ **Formulario de reservas** funcional
- ✅ **Diseño moderno y atractivo**
- ✅ **Optimizado para SEO**
- ✅ **Carga rápida**
- ✅ **Fácil de mantener y extender**

## 🎨 Estructura del Proyecto

```
El-Rincon-del-Tapeo-II/
├── index.html          # Página principal
├── css/
│   └── style.css      # Estilos principales
├── js/
│   └── script.js      # JavaScript interactivo
├── README.md          # Documentación
└── .gitignore         # Archivos a ignorar en Git
```

## 📱 Secciones del Sitio

1. **Header** - Logo y navegación principal
2. **Hero** - Imagen destacada y mensaje principal
3. **Especialidades** - Nuestras especialidades culinarias
4. **Carta** - Menú completo con precios
5. **Nosotros** - Historia y valores del establecimiento
6. **Contacto** - Información de contacto y horarios
7. **Reservas** - Formulario para reservar mesa
8. **Footer** - Información legal y redes sociales

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos y diseño responsive
- **JavaScript (ES6)** - Interactividad
- **Font Awesome** - Iconos
- **Google Fonts** - Tipografías (Poppins, Playfair Display)

## 🚀 Cómo Usar

### Desarrollo Local
1. Clona el repositorio:
   ```bash
   git clone https://github.com/comarni/El-Rincon-del-Tapeo-II.git
   ```

2. Abre `index.html` en tu navegador

3. Para desarrollo, puedes usar Live Server en VS Code

### Personalización

1. **Cambiar colores**: Edita las variables CSS en `css/style.css`
2. **Actualizar contenido**: Modifica `index.html`
3. **Cambiar imágenes**: Reemplaza las URLs en el HTML
4. **Añadir nuevas secciones**: Sigue la estructura existente
